# Konveyor – Extension Pack

The **Konveyor Extension Pack** installs everything needed to use Konveyor in Visual Studio Code.

It bundles the **Konveyor Core** extension together with all supported **language-specific extensions**, so you don’t need to install them individually.

## What’s included

- Konveyor Core
- Konveyor Java
- Konveyor JavaScript
- Konveyor Go
- Konveyor C sharp

## How it works

- All extensions are installed together
- Language extensions activate only when a matching project is detected
- Safe to install even if you use only one language

## Prefer a smaller install?

You can install a language-specific extension directly. Doing so will automatically install Konveyor Core.

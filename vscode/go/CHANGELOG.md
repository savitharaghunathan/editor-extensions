# Changelog

All notable changes to the "konveyor-go" extension will be documented in this file.

## [Unreleased]

### Added

- Initial release of the Konveyor Go extension
- Go language support for Konveyor migration and modernization analysis
- Integration with the official Go extension for enhanced language services
- Generic external provider for symbol resolution and code analysis
- Golang dependency provider for dependency analysis
- Automatic activation on Go projects (go.mod, go.sum files)
- Support for Go templates (gotmpl)
- Workspace file-based activation for improved project detection

### Changed

- Switch to document/symbol instead of workspace/symbol for improved performance
